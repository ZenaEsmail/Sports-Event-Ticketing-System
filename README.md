# Sports Event Ticketing System

## Overview

A System Analysis project for an online Sports Event Ticketing Platform developed for the Ministry of Youth and Sports.

The project covers business requirements, feasibility analysis, risk assessment, use case modeling, and data flow diagrams.

---

## Project Deliverables

- System Request
- Technical Feasibility
- Economic Feasibility
- ROI & Break-even Analysis
- Work Plan
- Staffing Plan
- Risk Assessment
- Use Cases
- Data Flow Diagram (DFD)

---

## Objectives

- User Registration
- Match Search
- Online Ticket Booking
- Secure Payment
- QR Ticket Generation
- Event Management
